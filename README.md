# analisador_excel.py
Um pequeno projecto para um analisador automatico de ficheiros excel, para ajudar nas tarefas repetitivas de ter de analisar ficheiros excel com algumas centenas ou milhares de linhas, obrigando a fazer comparações por vezes complicadas, o objectivo deste programa na versão 1.0 é de facilitar esse trabalho.
